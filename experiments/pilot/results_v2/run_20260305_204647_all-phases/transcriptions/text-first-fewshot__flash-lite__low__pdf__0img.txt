Le cas $\dim X = 0$ se décrit via les paradigmes du système $(\Pi_1(X) \to \Gamma, (p_i : \mathcal{C}_i \to \mathcal{E}, (k_i : \Pi_1(K_i) \to \Gamma)))$. Si dans $I = \emptyset$ — dans ce cas ce condition [unclear: "p injectif"] — donc se décrit simplement par la donnée d'un sous-groupe ouvert $\mathcal{E} \subset \Gamma$, i.e. d'une sous-extension finie $K$ de $\bar{K}/K$ CNB — assez bien sur $X' = \text{Spec } K$ les données [unclear: "fait"] sur $\Pi_1(X) \to \Pi_1(X')$ et la description — [unclear: "seulement"] ici on décrit une telle immersion de $\text{Gal}(\bar{K}/K) = \text{Hom}(\mathcal{E}, \mathcal{C}_k)$ de $\bar{K}/K$.

Le cas $\dim X = 1$ se traduit par le fait que $\mathcal{E} \to \Gamma$ n'est pas injectif — quand $I$, il peut être vide ou non dans ce cas. S'il est vide, la prescription se fait alors simplement en termes d'un homomorphisme de groupes finis $\mathcal{E} \to \Gamma$ là $\Gamma$ donne [unclear: "l'image"] = $\text{Gal}(\bar{K}/K) = \text{Aut}_{\Pi_1}(\Omega)$.
