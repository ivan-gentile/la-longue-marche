8. Réflexion taxonomique.

Le cas $\dim X = 0$ se traduit sur le paradigme du système $(T(\bar{k}), \mathcal{E}, \pi, \Gamma)$ par la condition $\pi$ injectif, donc se décrit simplement par la donnée d'un sous-groupe ouvert $\mathcal{E} \subset \Gamma$, i.e. d'une sous-extension finie $K$ de $\bar{K}/K$ (NB : aussi bien sur $X = \text{Spec } K$, le les données sur $X$ se bornant à la description d'une telle immersion $\mathcal{E} = \text{Hom}(K, \bar{K})$ des $\bar{K}/K$).

Le cas $\dim X = 1$ se traduit par le fait que $\mathcal{E} \to \Gamma$ n'est pas injectif — quand $I$, il peut être vide ou non dans ce cas. S'il est vide, la prescription se fait alors simplement en termes d'un hom. de groupes finis $\mathcal{E} \to \Gamma$ où $\Gamma$ donne le nom $= \text{Gal}(\bar{K}/K) = \text{Aut}_{\pi_e}(\Omega)$.
